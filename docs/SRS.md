# SRS (Software Requirements Specification)

## 1. 目的

Android 14以上向けのSSHクライアントを、ConnectBotをベースにKotlin + Composeで近代化し、以下を実現する。

- モバイルで崩れにくいターミナル表示
- Claude Code / Codex用ショートカット
- カスタムショートカット管理
- 日本語UI対応
- 個人利用前提・サーバー不要

---

## 2. スコープ

### 対象

- SSH接続（ホスト管理、known_hosts検証）
- ターミナル表示（ANSI対応）
- ショートカット機能（プリセット + カスタム）
- 日本語UI
- Android 14制約準拠

### 対象外

- クラウド同期
- Play Store公開
- SSHコア全面再実装

---

## 3. ユースケース

- ホスト作成 → 接続
- 接続中にショートカット実行
- カスタムショートカット編集
- JSONエクスポート/インポート
- 画面回転でも破綻しない

---

## 4. 機能要件

### 4.1 ホスト管理

- 作成/編集/削除
- hostname / port / username
- 認証: パスワード / 鍵
- 初回ホスト鍵確認必須

### 4.2 ターミナル

- 等幅フォント固定
- 文字サイズ変更
- コピー/ペースト
- 回転時PTYサイズ更新

### 4.3 ショートカット

- グローバル/ホスト別
- 複数行コマンド可
- 並び替え可能
- プレースホルダ展開
  - {{HOST}}
  - {{USER}}
  - {{DATE}}
  - {{DATETIME}}

---

## 5. 非機能要件

- minSdk = 34
- targetSdk = 34以上
- オフライン動作
- 鍵の平文外部保存禁止
